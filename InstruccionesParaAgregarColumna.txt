# Instrucciones para agregar la columna NumeroIdentidad a la tabla Clientes

Hay dos formas de agregar la columna NumeroIdentidad a la tabla Clientes:

## Opciu00f3n 1: Reiniciar la aplicaciu00f3n (Recomendado)

1. Detener la aplicaciu00f3n web si estu00e1 en ejecuciu00f3n
2. Iniciar nuevamente la aplicaciu00f3n

## Opciu00f3n 2: Ejecutar el script SQL manualmente

1. Abrir SQL Server Management Studio o cualquier otra herramienta de administraciu00f3n de SQL Server
2. Conectarse a la base de datos utilizada por la aplicaciu00f3n
3. Ejecutar el siguiente script SQL:

```sql

BEGIN
    IF NOT EXISTS (SELECT * FROM sys.columns WHERE object_id = OBJECT_ID('Clientes') AND name = 'NumeroIdentidad')
    BEGIN
        -- Agregar la columna NumeroIdentidad si no existe
        ALTER TABLE Clientes ADD NumeroIdentidad NVARCHAR(20) NULL;
        PRINT 'Columna NumeroIdentidad agregada correctamente a la tabla Clientes';
    END
    ELSE
    BEGIN
        PRINT 'La columna NumeroIdentidad ya existe en la tabla Clientes';
    END
END
ELSE
BEGIN
    PRINT 'La tabla Clientes no existe en la base de datos';
END
```

## Verificaciu00f3n

Para verificar que la columna se haya agregado correctamente, puedes ejecutar la siguiente consulta SQL:

```sql

WHERE TABLE_NAME = 'Clientes' AND COLUMN_NAME = 'NumeroIdentidad';
```

Si la consulta devuelve resultados, significa que la columna se ha agregado correctamente.
